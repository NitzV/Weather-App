# AuraWeather Dashboard 🌦️

A sleek, lightweight, full-stack weather application that delivers real-time weather forecasts and geocoding insights. The project bypasses traditional client-side limitations by utilizing a secure backend API proxy to fetch live atmospheric data streams.

---

## 🚀 Features

* **Secure Backend API Proxy:** Built with Python and Flask to manage outgoing network requests and handle live geocoding and forecast streams seamlessly.
* **Dynamic UI/UX:** A responsive frontend featuring modern glassmorphism design layouts.
* **Smart Theming:** Implements dynamic HSL background color shifts that automatically adapt based on current weather conditions and temperature changes.
* **Robust Error Handling:** Smoothly handles invalid city names or network drops without breaking the user experience.

---

## 🛠️ Tech Stack

* **Backend:** Python, Flask, Gunicorn
* **Frontend:** HTML5, CSS3 (Glassmorphism, Dynamic HSL), Vanilla JavaScript
* **Data Sources:** Open-Meteo API (Geocoding & Forecast streams)
* **DevOps & Hosting:** Git, GitHub, Render (Free Tier Web Service)
